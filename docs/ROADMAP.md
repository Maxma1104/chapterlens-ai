# Roadmap

## Now — validated MVP

- One chapter or long excerpt
- Paste and `.txt`
- Grounded structured review
- Auth, history, feedback, quota, monitoring, evaluation

## Next — only after real user evidence

1. Human-labeled entailment evaluation
2. DOCX and PDF parsing with page/paragraph anchors
3. Multi-chapter projects and incremental re-analysis
4. Cross-chapter character and timeline retrieval
5. Version comparison and exportable editorial report

## Explicit non-goals

- Autonomous manuscript rewriting
- Training on user manuscripts
- Replacing professional editorial judgment
- Claiming certainty from weak or missing evidence
